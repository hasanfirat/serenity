# how to clone project


- Download by ZIP

you can download by [ZIP](https://github.com/hasanfirat/serenity/archive/master.zip)

- or clone this repo

```
git clone https://github.com/hasanfirat/serenes.git
```
```
cd serenes
npm install
npm start
```
